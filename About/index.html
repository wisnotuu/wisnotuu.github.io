关于blog

一.blog的站点

本blog拟定设立两个站点

1.主站点

github.io原始站 or cloudflare加速站

  • 只是简单的GitHub Page，同时提供Cloudflare加速
2.实验性站点

onlineblog

  • 实验性站点将hexo全部托管到GitHub上（私人仓库），使用cloudflare page部署
  • 因为具有跨设备编辑的能力，所以所有blog会在第一时间发布到onlineblog中，包括实验性功能也会首先部署在onlineblog中（因此稳定性有所损失）
  • 也会在这里发布一些比较水的日常文章，并且不会二次发布到主站点上
  • 受限于cloudflare免费计划的部署次数限制，可能会多篇博客同时发布以减少每月部署次数

二. blog的基础

1.

此 blog 由 Hexo 驱动

2.

此 blog 采用了 butterfly 主题

3. and

此 blog 托管在 Github 上并且主站点部署在 Github Page上

4.

此 blog 主站点由 cloudflare 提供加速服务， onlineblog 部署在 Cloudflare Page 上

三. blog的目的

当然是为了无聊时摸鱼

为了记录些稀奇古怪的折腾~~（翻车）~~经历以及学习时的笔记
